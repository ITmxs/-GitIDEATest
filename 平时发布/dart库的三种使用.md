# Dart库

Dart中的库主要有三种:

- 1、我们自定义的库

```
import ‘lib/xxx.dart;
import ‘../../part-two/chapter3/enum.dart‘;
```

- 2、系统内置库

```
impont ‘dart:math
impont ‘dant:io‘;
import ‘dart:convert‘;
```

- 3、Pub包管理系统中的库([https://pub.dev](https://pub.dev/))

```
https://pub.dev/packages
https://pub.flutter-io.cn/packages
https://pub.dartlang.org/flutter
```
